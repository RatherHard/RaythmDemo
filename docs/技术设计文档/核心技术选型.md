# 核心技术选型

## 项目概述

本项目是一个基于 C++ 的跨平台音乐节奏游戏（音游），目标平台为 **Windows** 和 **Linux**。采用自研架构，以实现极致低延迟、精确的音画同步，以及高性能的谱面渲染能力。

---

## 技术选型总览

| 模块 | 技术选型 | 版本/说明 |
|------|---------|----------|
| **开发语言** | C++ | C++20 标准 |
| **构建系统** | CMake | 3.20+ |
| **开发环境** | VSCode | 配合 C++ 扩展 |
| **音频引擎** | miniaudio | 单头文件库，低延迟音频 |
| **图形渲染** | Vulkan | 现代跨平台图形 API |
| **窗口与输入** | SDL3 | 跨平台窗口、输入事件管理 |
| **JSON 解析** | nlohmann/json | Modern C++ JSON 库 |
| **编辑器 GUI** | Dear ImGui | 即时模式 GUI，配合 ImPlot 绘制波形 |

---

## 核心技术选型理由

### 1. 音频引擎：miniaudio

#### 选型理由

- **极低延迟**：直接封装各平台底层音频 API（Windows: WASAPI / Linux: ALSA/PulseAudio）
- **单头文件库**：集成简单，无外部依赖
- **高精度时钟**：提供 PCM 级别的精确时间戳，用于音画同步
- **跨平台**：完全跨平台，支持 Windows、Linux、macOS、iOS、Android
- **开源免费**：MIT 许可证，适合独立开发

#### 应用场景

- 实时音频播放与流式加载
- 音效触发（判定音效、按键反馈）
- 为谱面编辑器提供波形数据（解码 PCM）
- 精确的 DSP Time 计算，用于判定逻辑

---

### 2. 图形渲染：Vulkan

#### 选型理由

- **现代 API**：直接控制 GPU，性能极致
- **跨平台**：Windows（直接支持）、Linux（原生支持）
- **低驱动开销**：相比 OpenGL，驱动层开销更低
- **显式资源管理**：精确控制内存和同步，避免隐式状态切换
- **未来兼容性**：可通过 MoltenVK 支持 macOS，扩展性强

#### 应用场景

- 高性能 2D 渲染（谱面轨道、下落 Note、判定线）
- 高刷新率支持（144Hz/240Hz 显示器的无撕裂渲染）
- 着色器特效（Note 下落动效、击打特效、背景粒子系统）
- 多线程命令录制（利用 Vulkan 的命令缓冲机制优化渲染性能）

---

### 3. 窗口与输入：SDL3

#### 选型理由

- **跨平台输入抽象**：统一的键盘、鼠标、触摸输入接口
- **Raw Input 支持**：绕过操作系统的输入缓冲，降低判定延迟
- **事件驱动架构**：高效的事件轮询系统
- **与 Vulkan 集成**：SDL3 原生支持 Vulkan 窗口创建
- **成熟稳定**：Valve、osu! 等商业项目验证

#### 应用场景

- 创建窗口并管理 Vulkan Surface
- 处理键盘输入（K1-K7 音游键位）
- 实现全屏切换、分辨率调整
- 跨平台的高精度计时器（`SDL_GetPerformanceCounter`）

---

### 4. JSON 解析：nlohmann/json

#### 选型理由

- **现代 C++ 风格**：语法优雅，类似 Python 的 dict 操作
- **单头文件集成**：无需预编译，直接 `#include <nlohmann/json.hpp>`
- **类型安全**：编译期类型检查，减少运行时错误
- **性能充足**：对于谱面加载（仅在歌曲启动时解析一次），性能完全足够

#### 应用场景

- 解析谱面文件（.json 格式）
- 读取配置文件（settings.json）
- 保存游戏存档（成绩、解锁状态）
- 编辑器中的谱面序列化与反序列化

---

### 5. 编辑器 GUI：Dear ImGui + ImPlot

#### 选型理由

- **即时模式 GUI**：无需复杂的状态管理，代码即 UI
- **与渲染引擎深度集成**：直接在 Vulkan 渲染循环中绘制 UI
- **开发效率极高**：快速搭建编辑器界面（时间轴、属性面板、文件浏览器）
- **ImPlot 扩展**：专业的数据可视化库，用于绘制音频波形图
- **跨平台一致性**：UI 在所有平台上渲染完全一致（不依赖系统控件）

#### 应用场景

- 谱面编辑器（时间轴、Note 放置、BPM 编辑、判定线调整）
- 音频波形显示（使用 miniaudio 解码 PCM 数据，通过 ImPlot 绘制波形）
- 调试面板（实时显示帧率、音频延迟、判定偏移量）
- 设置界面（键位配置、视觉设置、音频设备选择）

## 依赖管理

### 第三方库获取方式

| 库名 | 获取方式 |
|------|---------|
| miniaudio | 单头文件，直接下载 `miniaudio.h` |
| Vulkan SDK | 官方安装器（LunarG SDK） |
| SDL3 | Git Submodule 或包管理器 |
| nlohmann/json | 单头文件或 CMake FetchContent |

| ImPlot | Git Submodule（ImGui 扩展） |

## 平台兼容性

| 平台 | 支持状态 | 备注 |
|------|---------|------|
| Windows 10/11 | ✅ 完全支持 | WASAPI 音频，Vulkan 1.3 |
| Linux (Ubuntu/Arch) | ✅ 完全支持 | ALSA/PulseAudio，Vulkan 1.3 |
| macOS | 🔄 未来支持 | 需通过 MoltenVK（Vulkan → Metal） |
| Android/iOS | 🔄 未来支持 | 需适配触摸输入和移动端音频 API |
