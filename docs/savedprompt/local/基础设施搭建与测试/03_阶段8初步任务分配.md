 2. 谱面模型与加载器

 新增：

 - include/Game/Chart.hpp
 - include/Game/ChartLoader.hpp
 - src/Game/ChartLoader.cpp

 推荐类型位于 Raythm::Game：

 - BeatPosition
   - 使用规约后的有理数表达谱面位置：std::int64_t numerator、std::int64_t denominator。
   - 支持相等与排序；例如 [1, 2, 4] 与 [1, 1, 2] 必须等价。
 - ChartMetadata
   - title、artist、creator、audioPath、offsetMilliseconds。
 - TimingPoint
   - BeatPosition beat、double bpm。
 - Note
   - BeatPosition beat、std::optional<BeatPosition> endBeat、int column。
   - 提供 isHold() const noexcept，本次只保存 hold 数据，不实现 hold 判定。
 - Chart
   - ChartMetadata meta、std::vector<TimingPoint> timingPoints、std::vector<Note> notes、int laneCount = 4。
 - ChartLoadOptions
   - int laneCount = 4。
 - ChartLoader
   - explicit ChartLoader(const ChartLoadOptions& options = {})
   - Chart loadFromJsonText(std::string_view jsonText) const
   - Chart loadFromFile(const std::string& filePath) const

 验证规则：

 - 顶层必须为 JSON object，且包含 meta、time、note。
 - meta.title/artist/creator/path 必须为 string，meta.offset 必须为 integer。
 - beat / endbeat 必须为三整数数组 [x, y, z]，满足 x >= 0、z > 0、0 <= y < z。
 - time 至少包含一个 BPM；每项 BPM 必须 finite 且 > 0。
 - 归一化排序后第一条 BPM 必须位于 beat zero；重复归一化 BPM beat 视为格式错误。
 - note.column 必须在 [0, laneCount - 1]。
 - 若存在 endbeat，必须严格晚于 beat。
 - time 与 note 可接受无序输入，loader 输出时排序。
 - 未知额外字段忽略。
 - 解析/验证失败使用 std::runtime_error 报错，错误消息包含字段上下文；loader 不直接输出日志。

 坐标约定：
